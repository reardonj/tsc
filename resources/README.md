# Resources

Members of the TSC have access to several communication channels and resources, enumerated below:

- This [repository].
- A [GitHub project], for tracking TSC tasks.
- The `tsc@typelevel.org` [mailing list]. 
- The `#tsc` channel on the [Typelevel Discord].
- [Tuple](tuple.md), for pair programming. (The linked document has additional instructions to get started.)
- A shared [Google Drive], that includes agendas/notes from work sessions. Any member of the TSC may add documents to the drive.
- A shared [Google Calendar]. Any member of the TSC may add events to this calendar, for example to schedule work sessions or other collaborative meetings.
